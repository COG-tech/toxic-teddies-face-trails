# Toxic Teddies: Face Trails

A collectible tracing puzzle game where every level map is built from the face of an original Toxic Teddy.

## Current milestone

The repository is being initialized with a playable Moldy Molly prototype, reusable SVG tracing engine, three toxic-drop lives, direction validation, completion reveal, and local progress saving.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The generated site is written to `dist/`.
