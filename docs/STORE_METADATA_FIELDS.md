# Store Metadata Fields

## Shared

- App name
- Short name/subtitle
- Primary category
- Secondary category
- Age/content rating
- Copyright
- Developer display name
- Privacy policy URL
- Support URL
- Marketing URL
- Release notes

## Apple App Store

- Subtitle
- Promotional text
- Description
- Keywords
- Review notes
- iPhone screenshots
- iPad screenshots when supported
- App Privacy answers
- Export compliance answers

## Google Play

- Short description
- Full description
- App icon
- Feature graphic
- Phone screenshots
- Tablet screenshots when supported
- Data Safety answers
- Ads declaration
- App access declaration
- Content rating questionnaire
