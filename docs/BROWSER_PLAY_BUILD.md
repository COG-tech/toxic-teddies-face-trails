# Playable Browser Build

GitHub Pages serves repository files directly from the configured branch. The application source imports JSON and Capacitor packages that require the Vite build step, so the raw source `index.html` cannot be used as the playable browser game.

The browser demo is generated into:

```text
play/
```

Playable URL:

```text
https://cog-tech.github.io/toxic-teddies-face-trails/play/
```

The repository-root page redirects GitHub Pages visitors to this bundled build. Native iOS and Android builds continue using the same verified `dist/` output through Capacitor.

Do not hand-edit files under `play/`. They are generated by `.github/workflows/build-browser-demo.yml` after validation and the Vite production build pass.
